# Find House Backend

You can find frontend of the application [here](https://github.com/ansaryergesh/find_house_frontend)

### Install, Run and Build program

```
$ bundle install --without production

$ rake db:setup

$ rails db:migrate

$ rails s -p 3002

```

## Built With

- Ruby on Rails API
- JWT Auth Token
